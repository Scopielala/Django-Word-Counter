# Django Word Counter

A simple and beginner-friendly **Django web application** that counts the number of words in a given text.  
This project demonstrates the fundamentals of backend web development — handling user input, text processing, and rendering output through Django templates.  

It’s ideal for anyone learning **Python, Django, and basic web development**.

---


---

## Overview
This Django project allows users to input any block of text, processes it on the backend, and returns the total word count instantly.  
It demonstrates:
- Django views and templates interaction
- Handling POST requests and form data
- Basic text manipulation using Python
- Dynamic rendering of content in the browser

---

## Features
Simple text form where users paste or type text  
Displays accurate word count dynamically  
Built using **Django’s template engine**  
Error handling for empty or invalid input  
Lightweight and easy to deploy  

---

## Tech Stack
- **Backend:** Python 3, Django
- **Frontend:** HTML, CSS (basic styling)
- **Database:** SQLite (default Django DB)
- **Environment Management:** Pipenv
- **Version Control:** Git & GitHub
